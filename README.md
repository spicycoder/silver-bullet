# Stencil Project

## Start new components project

```sh
npm init stencil
```

## Install Stencil as dev dependency

```sh
npm install
```

## Building the project

```sh
npm run build
```

> this generates a `dist` folder with minfied files

## Publishing the omponents

```sh
npm login
npm publish
```
